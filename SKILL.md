# 知乎热门话题 (Zhihu Hot Topics)

获取知乎网站每天的热门话题及其高赞回答，支持按分类筛选和自定义板块。

## 使用方式

### 第一步：确定获取方式

用户需要选择一种数据获取方式：

1. **用户自己提供API**（推荐）：用户自己调用知乎API获取数据，将API响应JSON传递给skill处理
2. **用户爬虫脚本**：用户自己编写爬虫脚本，skill使用脚本输出的JSON数据
3. **手动输入**：用户直接粘贴知乎热榜的JSON数据

如果用户没有数据获取方式，可以告知用户：
- 需要用户自己获取知乎数据（API或爬虫）
- skill负责数据解析、筛选和展示
- 不提供自动抓取功能（避免违法）

### 第二步：获取数据

根据用户选择的获取方式：

**方式1：用户提供API响应**
```
用户调用知乎API后，将JSON响应粘贴给skill
```

**方式2：执行用户脚本**
```
运行用户提供的脚本（如 python zhihu.py），获取输出JSON
```

**方式3：手动输入**
```
用户粘贴JSON数据格式的热榜内容
```

### 第三步：指定分类（可选）

可以指定筛选的分类：
- 经济
- 社会
- AI
- 科技
- 资源和工具
- 用户自定义（让用户输入感兴趣的板块）

如果不指定分类，默认返回全部分类的前10条热门话题。

### 第四步：处理和展示

对获取的数据进行：
1. 解析JSON数据
2. 按分类筛选（如果指定了分类）
3. 提取前10条热门话题
4. 获取每个话题下点赞最多的前3个回答
5. 格式化输出展示

## 数据格式要求

用户提供的JSON数据应包含以下字段：

```json
{
  "hot_list": [
    {
      "id": "123456",
      "title": "话题标题",
      "category": "AI",
      "answer_count": 100,
      "answers": [
        {
          "id": "answer_1",
          "author": "作者名",
          "content": "回答内容",
          "upvote_count": 500,
          "url": "回答链接"
        }
      ]
    }
  ]
}
```

如果数据格式不同，请告知用户需要调整字段映射。

## 脚本资源

如需数据获取脚本，见 `scripts/` 目录：
- `zhidu_api_example.py` - API调用示例脚本
- `parse_json.py` - JSON数据解析脚本

## 输出格式

输出内容：
1. 热门话题标题 + 分类标签
2. 回答数量
3. 前3个高赞回答：作者、点赞数、回答摘要、链接
4. 每个话题的消息卡片（可选）

## 注意事项

- 不提供自动抓取知乎数据的功能
- 只负责解析、筛选和展示用户提供的合法数据
- 如果用户需要爬虫脚本，需要用户自己编写或使用第三方合法工具
